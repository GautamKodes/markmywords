User flow:
1. START
2. Receive a word
3. Try to guess meaning with some hints like kind of verb, etc
4. Get the actual meaning with pronunciation and an example
5. Write 5-10 sentences using the word
6. Rewrite if more than 3 are incorrect 

Future Scope:
1. Login/Signup
2. Streaks

Tech:
Common frontend 
Python-lang-chain for LLM Integration
Words, hints, meanings stored in database or API or LLM
Java for backend if possible along with python
